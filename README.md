# Les listes chainées

### Liste chaînée simple

```bash
On a une liste qui contient des éléments dont ils ne stockent que seulement 
la valeur puis le pointeur du prochain liste
```

### Liste chaînée Double

```bash
On a une liste qui contient des éléments dont ils stockent 
une valeur puis le pointeur du prochain liste et le pointeur du liste
qui est devant lui.
```

### Liste chaînée Circulaire

```bash
On a pu voir dans les liste chaînées simple que chaque élément contient le pointeur
de son prédéceseur et quand c'est le dernier élément il va pointeur vers None ou null
mais dans liste chaînée circulaire le dernier élément a le pointeur du 1er élément.
```

### Liste doublement chaînée Circulaire

```bash
On a pu voir dans les liste chaînées double que chaque élément contient le pointeur
de son prédéceseur et le pointeur de l'élément avant lui, quand c'est 
le dernier élément il va pointeur vers le tête de liste.
```

### Pile

```bash
On empile des éléments à partir de haut, mais on depile les éléments à partir
de haut. (LIFO)
```

### Queue

```bash
On empile des éléments à partir de bas, mais on depile les éléments à partir
de haut. (FIFO)
```
