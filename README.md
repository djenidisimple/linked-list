# Les listes chainées

### Liste chaînée simple

```bash
On a une liste qui contient des éléments dont ils ne stockent que seulement 
la valeur puis le pointeur du prochain liste
```

### Liste chaînée Double

```bash
On a une liste qui contient des éléments dont ils stockent 
une valeur puis le pointeur du prochain liste et le pointeur du liste
qui est devant lui.
```
